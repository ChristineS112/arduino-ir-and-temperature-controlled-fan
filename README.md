# arduino-ir-and-temperature-controlled-fan

This project controls a fan blade attached to a DC motor using an IR remote and a temperature sensor. If the temperature rises above 25Â°C, the motor turns on automatically. The fan can also be triggered manually using the remote. System information, including temperature readings and fan status, is displayed on a 16x2 LCD.



**Hardware Components**

- Arduino (Uno)
- IR Receiver + IR Remote
- DC Motor + Fan Blade
- L293D Motor Driver
- DHT11 Temperature & Humidity Sensor
- 16x2 LCD Display
- Potentiometer
- Power Supply Module
- 9V battery
- Breadboard & jumper wires

All components can be found in the ELEGOO UNO Project Super Starter Kit [here](https://www.amazon.ca/dp/B01D8KOZF4)
